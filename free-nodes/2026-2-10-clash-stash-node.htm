从零开始的Quantumult之旅

获取与安装

Quantumult的获取途径简单直接： 1. 打开iOS设备的App Store 2. 搜索"Quantumult"（注意区分正版与山寨应用） 3. 完成购买与下载（部分功能可能需要内购解锁）

安装完成后，首次打开应用时，您会看到一个简洁但功能分明的界面。与许多同类工具不同，Quantumult没有用复杂的技术术语吓退用户，而是通过清晰的菜单结构引导初学者逐步完成配置。

核心配置详解

服务器添加是使用Quantumult的第一步。您可以通过两种方式完成：

  1. 手动添加：

    • 进入"节点"页面
    • 点击右上角"+"号
    • 根据您的协议类型（SS/V2Ray等）填写服务器信息
    • 保存并测试连接

  2. 订阅链接（推荐方式）：

    • 进入"配置"→"订阅"
    • 添加您的订阅链接
    • 设置自动更新频率（建议24小时）
    • 保存后Quantumult会自动获取节点列表

值得注意的是，Quantumult支持多订阅源管理，这意味着您可以同时使用多个服务提供商，并根据网络状况智能切换。

规则引擎：Quantumult的灵魂所在

如果说代理功能是Quantumult的身体，那么规则系统就是它的灵魂。通过精心配置规则，您可以实现：

  • 国内外流量智能分流（直连国内网站，代理国外资源）
  • 特定应用强制走代理（如Twitter、YouTube）
  • 广告屏蔽与隐私保护
  • 根据网络状况自动切换节点

配置规则的基本步骤： 1. 进入"规则"页面 2. 选择"添加规则" 3. 设置匹配条件（域名、IP、URL关键词等） 4. 指定策略（直连、代理、拒绝等）

高级用户还可以导入现成的规则集，或基于GitHub上的开源规则进行二次开发。一个精心调校的规则系统可以显著提升网络体验，同时减少不必要的流量消耗。

脚本功能：解锁无限可能

Quantumult的脚本支持是其最强大的特性之一。通过编写或导入JavaScript脚本，您可以实现：

  • 自动切换最优节点

常见脚本应用场景包括： - 流媒体解锁（绕过区域限制） - 特定网站自动签到 - 网络质量监控与报警 - 自定义广告过滤规则

对于非技术用户，Quantumult的脚本仓库和社区分享提供了大量现成解决方案，只需简单导入即可享受这些高级功能。

性能优化与疑难排解

要让Quantumult发挥最佳性能，有几个关键点需要注意：

  1. 节点选择策略：

    • 启用"延迟测试"功能定期评估节点质量
    • 设置故障转移规则，当主节点不可用时自动切换
    • 根据不同用途（浏览/视频/下载）配置不同节点组

  2. 资源消耗控制：

    • 调整"并发连接数"（默认32，可根据设备性能调整）
    • 合理设置DNS缓存时间
    • 关闭不必要的日志记录

  3. 常见问题解决：

    • 连接失败：检查订阅是否过期、节点信息是否正确
    • 速度慢：尝试切换协议（如从SS改为V2Ray的WS+TLS）
    • 规则不生效：检查规则顺序（从上到下匹配）和语法错误

FAQ

如何配置 Sing-box 使用 DoH 加密 DNS？
在 dns 模块中添加类型为 doh 的上游，如 `https://1.1.1.1/dns-query`，并指定 bootstrap 地址。启用加密 DNS 可防止 DNS 泄露并加快国际解析速度，提高整体隐私保护。
Nekoray 与 Nekobox 有什么不同？
Nekoray 是一款基于 Qt 的跨平台代理前端，而 Nekobox 更偏向移动端简化使用。两者均支持 V2Ray/VLESS/Trojan 协议，但 Nekobox 提供了更轻量的界面和更快的连接启动速度。
Sing-box 的自定义 outbound 功能有什么用？
Sing-box 允许用户自定义多个 outbound 出口，并通过路由规则选择性输出。例如可让流媒体走特定节点，而其他网站使用备用节点，从而实现精准分流。
SSR 如何选择合适混淆提高稳定性？
选择混淆方式时应根据网络环境和封锁程度，例如 tls1.2_ticket_auth 可模拟 HTTPS 流量。合理组合协议和混淆可提升节点稳定性和抗封锁能力，确保长期科学上网体验。
Hysteria2 多路复用对短连接性能有何提升？
多路复用允许多个短连接共享同一通道，减少握手延迟和资源消耗。适合高频短连接场景，如游戏和即时通讯，可提高吞吐量和连接稳定性，优化整体网络体验。
Sing-box 精确端口分流如何提升游戏连接？
Sing-box 可按端口设置流量分流规则，将游戏流量分配到低延迟节点。结合策略组使用，可保证游戏连接稳定，减少延迟和掉线，提高流畅度和网络体验。
VMess 节点无法连接时应如何排查？
首先检查服务器是否在线，其次查看端口、防火墙和 TLS 配置是否正确。还可以通过日志观察 handshake 是否成功，或尝试更换传输协议如 WS、gRPC 来确认问题来源。
QuantumultX 如何使用分流策略实现广告屏蔽？
QuantumultX 可通过规则匹配广告域名或路径，将请求重定向或直接阻断。结合策略组，可实现自动屏蔽网页和应用广告，提高用户浏览体验。
Shadowrocket 节点延迟排序如何提高稳定性？
Shadowrocket 测试节点延迟和丢包率，将低延迟节点优先使用。结合策略组和规则文件，可保证网页、视频和游戏流量通过最优节点，提高连接稳定性和访问速度。
WinXray 的主要特点是什么？
WinXray 是一款基于 V2Ray 核心的 Windows 客户端，界面直观，支持 VMess、VLess、Trojan、Shadowsocks 等多协议。它提供一键导入订阅、节点测速、自动分流等功能，是适合初学者和进阶用户的图形化代理工具。
